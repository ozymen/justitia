Language specific notes

C

  • C code is compiled with gcc, by default without the --ansi flag. So GNU extensions are enabled.

Java

  • The name of the main class is detected automatically:
    • Only classes with a main method are considered.
    • If there is a class with the same name as the source file, then that one is used.
    • In case of ambiguity an error is reported.

Matlab

  • ".m" files are run using octave instead of matlab. Octave should be compatible, but there may be some issues with uncommon functions.
  • A .m file can be either a function or a stand alone script.
  • For running stand alone scripts, an empty file called "run-script.in" should be made. The script will be run and the output compared.
  • For running functions, the info file must include the attribute 
    compiler: matlab-function
    Now multiple test case inputs can be made. Such an input file should be a script that calls the function to be tested. For example: 
    function_to_test(1,2)
function_to_test(1,3)
function_to_test(2,4)
    This script is then run in the same directory as the tested function, and the output is compared.
