Q: Some submissions fail to judge correctly, I don't know what happens.
You can manually compile and judge submissions from the command line.

  • Log in to the justitia server.
  • Go to the backend directory.
  • Execute ./try_to_judge_single_submission <submissionid>.
This judges the submission with all debug outputs on, and doesn't delete temporary files.
These temporary files will contain the output of the compiler and runner.

If you have any question not answered here, you can contact the developer.
