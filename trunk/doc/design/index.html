Design of the program

Introduction

Justitia is written in PHP5.

Directories

lib/
Contains the classes of the program, one class per file. bootstrap.inc sets up autoloading of classes.
frontend/
Files visble from the website. Each .php file is a different page.

Pages intended only for administrators are called admin_*.php. Pages which are intended to be called from javascript as part of the gui are called ajax_*.php.

backend/
The judge daemons.
config/
Configuration files, see installation instructions.
install/
Database construction script.
doc/
This documentation.
